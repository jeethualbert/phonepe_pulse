**📊 PhonePe Pulse Data Visualization & Exploration**


**📌 Project Overview**

This project analyzes the PhonePe Pulse dataset to understand the growth of digital payments across India.
The dataset contains information about transactions, insurance usage, and user engagement across states, districts, and pincodes. 

The project performs data extraction, transformation, storage, and visualization to generate insights into digital payment adoption trends in India.

An interactive dashboard was built using Streamlit to explore the data through dynamic charts and maps.


**🎯 Problem Statement**

The objective of this project is to:

* Extract large-scale PhonePe transaction data from JSON files
* Transform and structure the data for analysis
* Store the data in a relational database
* Build interactive visualizations to analyze digital payment trends
* Identify patterns in transactions, user engagement, and insurance usage across India


**🛠️ Technologies Used**

| Technology | Purpose                      |
| ---------- | ---------------------------- |
| Python     | Data processing and analysis |
| Pandas     | Data manipulation            |
| MySQL      | Data storage                 |
| Streamlit  | Interactive dashboard        |
| Plotly     | Data visualization           |
| JSON       | Source data format           |


**⚙️ Project Workflow**

1. Extract data from PhonePe Pulse JSON files.
2. Transform data using Pandas (cleaning & structuring).
3. Load data into MySQL database.
4. Visualize insights using Streamlit and Plotly charts.


**📊 Features**

State-wise transaction analysis
District-level insights
Insurance transaction trends
User engagement analysis
Interactive dashboards with charts and maps


**📊 Visualizations Used**

The dashboard includes multiple interactive visualizations:

* Bar Charts
* Pie Charts
* Line Charts
* Choropleth Maps
* District-wise comparisons

These visualizations help identify:

* Transaction growth trends
